# 사고원리 매핑 — Directed Mutation용

실패 패턴을 분석할 때, 아래 매핑에서 관련 사고원리를 찾아 스킬 프롬프트에 1-2문장으로 주입한다. 트리거 자체를 넣는 게 아니라, 그 안의 핵심 원리만 추출한다.

---

## 실패 유형 → 사고원리 매핑

| 실패 유형 | 증상 | 적용할 사고원리 | 주입 예시 |
|-----------|------|----------------|----------|
| **누락/불완전** | 필수 요소 빠짐, 부분만 다룸 | 절대자(부분해→완전해→빠진부분 탐색) | "출력 전 모든 필수 요소가 포함됐는지 전수 확인하라" |
| **구조 혼란** | 순서 뒤섞임, 논리 흐름 끊김 | 백본(결론먼저→역방향→필수논거만) | "결론을 먼저 놓고 역방향으로 필요 논거를 배치하라" |
| **과잉/비대** | 불필요 내용 포함, 핵심 묻힘 | 스켈레톤(비본질 제거→뼈대만) | "각 요소가 핵심 목적에 기여하는지 검증하고 미기여 요소를 제거하라" |
| **표면적 분석** | 깊이 없음, 현상 나열만 | 제1원리(유추 차단→환원불가 명제까지 분해→재조립) | "표면 현상이 아닌, 근본 원인까지 분해하여 설명하라" |
| **단일 관점** | 한 측면만 다룸, 대안 부재 | 홈즈(복수가설→증거대조→소거) | "최소 2개 대안을 제시하고 증거로 비교하라" |
| **모호한 판단** | 결론 불명확, 양비론 | 오컴(경쟁가설 병렬→가정최소인 쪽 선택) | "복수 해석이 있으면 가정이 가장 적은 쪽을 선택하고 근거를 밝혀라" |
| **맥락 무시** | 상황/제약 조건 반영 안 됨 | 모순기술(욕구·제약 명시→분리원리→최적 도출) | "목표와 제약을 명시하고, 둘 다 만족하는 해법을 도출하라" |
| **좁은 시야** | 부분 최적화, 전체 영향 무시 | 줌(미시↔거시 자유전환) | "개별 요소뿐 아니라 전체 시스템에 미치는 영향을 검토하라" |
| **비현실적 제안** | 실행불가능, 자원 무시 | 맥가이버(제약→가용자원 재해석→창의적 조합) | "현재 가용한 자원과 제약 내에서 실행 가능한 방안을 제시하라" |
| **리스크 간과** | 실패 가능성 미고려 | 프리모르템(실패 전제→역추적→치명원인 Top3→차단책) | "이 방안이 실패할 수 있는 상위 3개 원인과 대비책을 포함하라" |
| **분류 부재** | 항목이 뒤섞여 있음, MECE 아님 | 엄브렐러(전수→MECE분류→상위개념 명명) | "모든 항목을 빠짐없이 나열한 뒤 겹침 없이 분류하라" |
| **인과 오류** | 상관을 인과로, 논리 비약 | 베이지안(사전확률→새증거→사후확률 갱신) | "주장의 근거가 되는 증거 강도를 명시하고, 확실성 수준을 밝혀라" |
| **유사성 남용** | 부적절한 비유, 타 사례 무비판 적용 | 아날로지(구조적 패턴 추출→매핑→적용가능성 검증) | "비유나 사례를 들 때 구조적 유사성과 차이점을 모두 밝혀라" |
| **2차 효과 무시** | 직접 효과만, 파급 효과 누락 | 시뮬레이션(피드백루프→n차효과→지렛점 특정) | "1차 효과뿐 아니라 2-3차 파급 효과를 추적하라" |
| **축별 분석 부재** | 다축 분석 필요한데 단일축만 | 연역수렴(축별 독립연역→교차연역→상위인사이트) | "최소 3개 축에서 독립 분석 후 교차점을 도출하라" |
| **순서 꼬임** | 단계가 논리적 순서 아님 | 타임스톤(인과·선후관계→시간축 정렬→순서 교정) | "각 단계의 선후 의존관계를 파악하고 올바른 순서로 재배열하라" |

---

## 사용법

1. 실패한 출력을 읽고 위 표에서 가장 가까운 "실패 유형"을 찾는다
2. 해당 "주입 예시"를 참고하여 스킬의 맥락에 맞게 1-2문장으로 변형한다
3. 스킬 프롬프트의 적절한 위치에 삽입한다 (관련 지시 근처)
4. 한 번에 하나의 원리만 주입한다 — 복수 원리를 동시에 넣으면 어떤 게 효과인지 알 수 없다

## 주의사항

- 트리거 이름(홈즈, 오컴 등)을 스킬 프롬프트에 직접 쓰지 마라. 내부 용어다.
- 원리를 그대로 복사하지 말고, 해당 스킬의 도메인 언어로 번역하라.
- 주입 문장이 기존 지시와 모순되면 기존 지시를 수정하거나 주입을 재고하라.
